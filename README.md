# Dynamic Decoding Utility

This repo includes a runnable **Xcode playground** that demonstrates decoding deeply nested JSON using `JSONDecoder.decode(_:from:path:)`.

## How to view the docs page

1. Open the project in Xcode.
2. In the navigator, open `DynamicDecoding.playground` → **Overview** page.
3. In the menu bar, choose **Editor → Show Rendered Markup** to see the formatted explanation.
4. Press **Run** on the playground to execute the examples and watch console output.
